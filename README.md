A simple discord bot for the RL Foxes discord server.

## Setup

### Run with Docker

```bash
# Build and run with Docker Compose
docker-compose up -d

# View logs
docker-compose logs -f

# Stop the bot
docker-compose down
```

### Or Run Locally (without Docker)

```bash
# Install dependencies
pip(3) install -r requirements.txt

# Run the bot
python main.py
```
